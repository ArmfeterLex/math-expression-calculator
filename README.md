# Math Expression Calculator

Калькулятор математических выражений на C# Windows Forms с выбором формулы из выпадающего списка.

## Функциональность

- Выбор выражения из ComboBox (5 примеров)
- Ввод переменных (x, y, z в зависимости от формулы)
- Автоматический расчет по выбранной формуле
- Очистка полей при загрузке формы

## Доступные выражения

### Пример 1
`((2 * cos(x - π/6)) / (0.5 + sin²(y))) * (1 + (z²/3 - z²/5))`

### Пример 2
`(x + y) * (x - y)` — разность квадратов

### Пример 3
`x³ + y³` — сумма кубов

### Пример 4
`x * sin(y) + cos(x)`

### Пример 5
`(x + y + z) / 3` — среднее арифметическое трех чисел

## Интерфейс

- comboBox1 — выбор примера (1-5)
- textBox1 — переменная x
- textBox2 — переменная y
- textBox3 — переменная z (для примеров 1 и 5)
- textBox4 — результат
- button1 — вычислить

## Пример работы

**Пример 5 (среднее арифметическое):**
Выбор "Пример 5", ввод: x=10, y=20, z=30 → результат 20

## Куда можно встроить

- В образовательные приложения по математике
- В инженерные калькуляторы
- В демонстрацию математических формул
- В системы автоматических расчетов

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

В примере 1 используется число π (Math.PI) и тригонометрические функции. Все переменные должны быть числами, иначе программа вызовет исключение.
